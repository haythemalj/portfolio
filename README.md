## 📋 Setup Instructions

### Database Setup
- Database: `portfolio`
- Username: `root`
- Password: (empty)
- Host: `127.0.0.1`

### Installation

```bash
# 1. Clone and navigate
cd C:\laragon\www\Portfolio

# 2. Install dependencies
composer install
npm install

# 3. Setup database
php artisan migrate
php artisan db:seed

# 4. Build assets
npm run dev

# 5. Start server
php artisan serve
```

Visit: `http://localhost:8000`
